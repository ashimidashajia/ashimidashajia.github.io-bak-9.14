VIM 复制内容到剪贴板 剪贴板复制到VIM

VIM 复制内容到剪贴板 剪贴板复制到VIM

标签： linux常用命令

这里我直接给出操作过程，方便新手。

1. ESC进入命令模式
2. v 进入块操作模式
3. 移动光标选中要复制的代码块
4. 输入   “+y

这三个字符，酱紫你的内容就被复制到系统的剪贴板了，就可以跟平常一样使用了。
同理,可以把系统剪贴板中的内容复制到VIM中，使用 “+p这样就可以了。

重新挂载system分区，使其具有写权限

重新挂载system分区，使其具有写权限

标签： 刷机

首先保证自己具有root权限

adb shell
su //切换到root用户
mount -o remount,rw /system

或者

adb root
adb shell mount -o remount rw /system

这样执行成功的话，就可以在system分区任意写东西了。

linux下利用rsync快速同步即拷贝数据

linux下利用rsync快速同步即拷贝数据

标签（空格分隔）：rsync同步数据

rsync 简介

rsync（remote synchronize）是一个远程数据同步工具，可通过 LAN/WAN快速同步多台主机之间的文件。也可以使用rsync 同步本地硬盘中的不同目录。
rsync 是用于替代 rcp 的一个工具，rsync 使用所谓的rsync算法 进行数据同步，这种算法只传送两个文件的不同部分，而不是每次都整份传送，因此速度相当快。
